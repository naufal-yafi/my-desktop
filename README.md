# Minimal installation OS Server to OS Daily Use.

![screenshot](./wallpaper/wallhaven-1kv23v_1366x768.png)

```sh
sh install --base-system --dev-tools --ms-edge --firefox --chrome
```

```sh
sudo reboot
```

## Show All File Error Log

```sh
sh install --log
```

## Show Specific File Error Log

```sh
sh install --log={timestamp}.log
```

Example: `sh install --log=log/25-12-2025_08.47.log`
